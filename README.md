# Pathways Explorer

An interactive navigation instrument for "Pathways: The Future of Law to 2040" — the scenario project convened by The LegalTech Fund, Harvey, and Law.com. Three questions, eight possible worlds, four built in detail.

This is a single self-contained `index.html`: vanilla JavaScript, no build step, no dependencies, no external calls. Open it by double-click, or serve it as a static file (e.g., GitHub Pages).
